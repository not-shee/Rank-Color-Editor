# Rank-Color-Editor
Tool that allows you to change the colors of ranks on the osu! website.

## How to install
1. Install Tampermonkey/Greasemonkey and make sure it's enabled. 
1. Go to https://github.com/not-shee/Rank-Color-Editor/raw/main/RankColorEditor.user.js
3. Tampermonkey/Greasemonkey will automatically open a tab, prompting you to install the userscript. Install it.
4. You're done. You can now edit the rank colors on the osu! website

## Tutorial
Once you open https://osu.ppy.sh/ you will see two new buttons appear at the top right of the page. 
![alt text](https://files.catbox.moe/66gmqk.png)

The settings button opens a settings menu where you can select colors for every rank tier.
![alt text](https://files.catbox.moe/8xjzy3.png)

The reset button resets ALL rank colors to default and refreshes the webpage.

Enjoy the script!!!
